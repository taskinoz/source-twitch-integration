# Items

## How to use
With Bits:
  You can use bits to give the streamer items e.g ``PogChamp20 give tone``

Without Bits:
  If you used !claim you can now use ``!give tone``

## Items

|tone|The gun loadout from Tone|
|scorch|The gun loadout from Scorch|
